# Tank Upgrade Tree Analysis - Diep.io Style RPG Game

## Complete Upgrade Tree Structure

### Tier 0 (Level 1)
- **Basic** - Starting tank

### Tier 1 (Level 15)
From **Basic**:
- **Twin** - Double barrels
- **Sniper** - Long-range barrel
- **Machine Gun** - Rapid fire trapezoid barrel
- **Flank Guard** - Front and rear barrels
- **Smasher** - No barrels, hexagon body

### Tier 2 (Level 30)

#### From **Twin**:
- **Triple Shot** - Three angled barrels
- **Quad Tank** - Four directional barrels (also from Flank Guard)
- **Twin Flank** - Double front + double rear (also from Flank Guard)

#### From **Sniper**:
- **Assassin** - Longer, thinner barrel
- **Overseer** - Drone controller with 8 triangular drones
- **Hunter** - Dual-layered barrel
- **Trapper** - Trap deployer

#### From **Machine Gun**:
- **Destroyer** - Large caliber barrel
- **Gunner** - Four smaller barrels

#### From **Flank Guard**:
- **Tri-Angle** - Forward + two rear thrusters
- **Quad Tank** - Four directional barrels (also from Twin)
- **Auto 3** - No player barrels, 3 auto turrets
- **Twin Flank** - Double front + double rear (also from Twin)

#### From **Smasher**:
- **Crusher** - Enhanced hexagon body

### Tier 3 (Level 45)

#### From **Triple Shot**:
- **Penta Shot** - Five angled barrels
- **Triplet** - Three straight barrels
- **Spread Shot** - Eleven spread barrels
- **Starfall Arsenal** (Tier 4) - Seven enhanced barrels

#### From **Quad Tank**:
- **Octo Tank** - Eight directional barrels
- **Auto 5** - Five auto turrets (also from Auto 3)
- **Orbital Array** (Tier 4) - Enhanced octo with satellites

#### From **Assassin**:
- **Ranger** - Extreme range barrel
- **Stalker** - Invisibility sniper

#### From **Overseer**:
- **Necromancer** - Square body, 34 square drones
- **Manager** - Single spawner, invisibility, 16 drones
- **Factory** - Minion spawner, 12 minion drones
- **Battleship** - Four spawners, 32 drones (also from Twin Flank)
- **Overlord** - Four spawners, 8 drones

#### From **Hunter**:
- **Predator** - Triple-layered barrel
- **Streamliner** - Five stream barrels

#### From **Destroyer**:
- **Annihilator** - Massive barrel
- **Hybrid** - Destroyer + drone spawners
- **Skimmer** - Destroyer variant
- **Rocketeer** - Destroyer variant

#### From **Tri-Angle**:
- **Booster** - Enhanced thruster configuration
- **Fighter** - Side cannons + thrusters

#### From **Crusher**:
- **Spike** - Spiky hexagon, 6 spikes
- **Landmine** - Hexagon with invisibility
- **Auto Smasher** - Hexagon + auto turret

#### From **Gunner**:
- **Auto Gunner** - Gunner + rear auto turret
- **Gunner Trapper** - Front gunner + rear trapper

#### From **Machine Gun**:
- **Sprayer** - Wide trapezoid barrel

#### From **Trapper**:
- **Mega Trapper** - Larger traps
- **Overtrapper** - Trapper + drone spawners

#### From **Twin Flank**:
- **Triple Twin** - Six barrels in three pairs
- **Battleship** - Four spawners, 32 drones (also from Overseer)

#### From **Auto 3**:
- **Auto 5** - Five auto turrets (also from Quad Tank)

### Tier 4 (Level 60) - Evolution Tanks

- **Siegebreaker** (from Annihilator, Skimmer, Rocketeer) - Massive siege cannon
- **Aegis Vanguard** (from Spike, Landmine, Auto Smasher) - Enhanced spiked defense
- **Mirage** (from Stalker) - Advanced invisibility with decoys
- **Starfall Arsenal** (from Penta Shot, Triplet, Spread Shot) - Seven enhanced barrels
- **Orbital Array** (from Octo Tank, Auto 5, Triple Twin) - Enhanced multi-barrel with satellites
- **Phase Sentinel** (from Ranger, Predator, Streamliner) - Advanced long-range
- **Gravemind Regent** (from Necromancer, Manager) - Enhanced necromancer
- **Armada Colossus** (from Factory, Battleship, Overlord, Auto 5) - Mega carrier
- **Citadel Shaper** (from Mega Trapper, Overtrapper, Gunner Trapper) - Advanced fortress builder
- **Cyclone Bulwark** (from Auto Gunner, Sprayer) - Spinning defense system
- **Fusion Hydra** (from Hybrid) - Multi-headed destroyer-drone hybrid
- **Velocity Reaver** (from Fighter, Booster) - Ultimate speed tank

### Tier 5 (Level 75) - Mythic Tanks

- **Obelisk** (from Phase Sentinel, Citadel Shaper, Mirage) - Laser pylon builder
- **Tempest** (from Cyclone Bulwark, Orbital Array) - Vortex generator
- **Bulwark Prime** (from Aegis Vanguard) - Ultimate defense
- **Cataclysm Engine** (from Siegebreaker) - Apocalyptic siege weapon
- **Astral Regent** (from Gravemind Regent, Armada Colossus) - Warp-capable carrier
- **Ion Vanguard** (from Velocity Reaver, Fusion Hydra, Starfall Arsenal) - Energy lance fighter

### Tier 6 (Level 90) - Ascended Tanks

- **Riftwalker** (from Astral Regent) - Teleporting drone master
- **Catalyst** (from Ion Vanguard) - Combo-building fighter
- **Bastion Eternal** (from Bulwark Prime) - Overshield fortress
- **Doomsday Harbinger** (from Cataclysm Engine) - Apocalypse cannon
- **Prism Archon** (from Obelisk) - Beam refraction master
- **Maelstrom Sovereign** (from Tempest) - Storm lord

## Tank Archetypes by Category

### Bullet Spammers
- **Basic Branch**: Basic → Twin → Triple Shot → Penta Shot → Starfall Arsenal
- **Machine Gun Branch**: Machine Gun → Gunner → Auto Gunner → Cyclone Bulwark
- **Spread Branch**: Triple Shot → Spread Shot → Starfall Arsenal

### Snipers
- **Precision Branch**: Sniper → Assassin → Ranger → Phase Sentinel
- **Stealth Branch**: Assassin → Stalker → Mirage → Obelisk
- **Multi-Layer Branch**: Sniper → Hunter → Predator/Streamliner → Phase Sentinel

### Destroyers
- **Pure Power**: Machine Gun → Destroyer → Annihilator → Siegebreaker → Cataclysm Engine → Doomsday Harbinger
- **Hybrid Variants**: Destroyer → Hybrid → Fusion Hydra → Ion Vanguard → Catalyst

### Drone Masters
- **Overseer Line**: Sniper → Overseer → Overlord → Armada Colossus → Astral Regent → Riftwalker
- **Necromancer Line**: Overseer → Necromancer → Gravemind Regent → Astral Regent → Riftwalker
- **Battleship Line**: Overseer → Battleship → Armada Colossus → Astral Regent → Riftwalker

### Speed Tanks
- **Fighter Line**: Flank Guard → Tri-Angle → Fighter → Velocity Reaver → Ion Vanguard → Catalyst
- **Booster Line**: Tri-Angle → Booster → Velocity Reaver → Ion Vanguard → Catalyst

### Smashers
- **Spike Line**: Basic → Smasher → Crusher → Spike → Aegis Vanguard → Bulwark Prime → Bastion Eternal
- **Stealth Line**: Crusher → Landmine → Aegis Vanguard → Bulwark Prime → Bastion Eternal

### Trappers
- **Fortress Line**: Sniper → Trapper → Mega Trapper → Citadel Shaper → Obelisk → Prism Archon
- **Hybrid Line**: Trapper → Overtrapper → Citadel Shaper → Obelisk → Prism Archon

### Auto Tanks
- **Auto Line**: Flank Guard → Auto 3 → Auto 5 → Orbital Array → Tempest → Maelstrom Sovereign

### Multi-Barrel Tanks
- **Octo Line**: Twin/Flank Guard → Quad Tank → Octo Tank → Orbital Array → Tempest → Maelstrom Sovereign

## Summary
The current upgrade system is well-developed but has some gaps, particularly in early-tier variety and unique high-tier paths. The synergy system for Tier 4+ tanks creates interesting stat optimization choices, encouraging different build strategies for the same tank class. The drone masters and speed tanks have the most complete upgrade trees, while machine gun and smasher branches could use expansion at lower tiers.
